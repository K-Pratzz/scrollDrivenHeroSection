# Scroll-Driven Hero Section Animation

A smooth, scroll-based hero section with letter-spaced headline, animated statistics, and a car that moves upward naturally as you scroll — built as a frontend assignment.

Live Demo: [https://your-live-link.netlify.app](https://your-live-link.netlify.app)  
*(Replace with your actual Netlify link after deployment)*

## ✨ Features

- Clean hero section that takes full first screen
- Premium letter-spaced headline: **W E L C O M E I T Z F I Z Z**
- Statistics cards that animate in one by one on page load
- Smooth scroll-driven car animation (parallax effect using `transform`)
- Fully responsive design
- No heavy libraries — only basic React + CSS
- Performant and smooth 60fps animation
- Extra scroll sections to test the car movement

## 🎯 Assignment Requirements Fulfilled

- Hero section layout with headline and statistics
- Initial load animation (fade + slide)
- Scroll-based interaction (car moves based on scroll position)
- Uses only `transform` for smooth performance
- Built with vanilla HTML, CSS, and JavaScript inside React

## 🛠️ Tech Stack

- **React** (Vite)
- **HTML**
- **CSS** (plain CSS – no Tailwind)
- **JavaScript**
